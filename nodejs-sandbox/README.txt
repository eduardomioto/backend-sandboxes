## Instalation

- Windows: https://nodejs.org/dist/v6.9.1/node-v6.9.1-x64.msi
- Linux:
	sudo apt-get update
	sudo apt-get install node

## Execution

1 - Choose a node file (ex: server-creation.js), and run:
	node server-creation.js

## Modules Explanation [Basic]
