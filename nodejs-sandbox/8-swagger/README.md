### Swagger and NodeJS

##### Installing

* `npm install -g swagger`

##### Quick Ref
* `swagger project create movie-collection`
* `swagger project start movie-collection`
* `swagger project edit movie-collection`

Mock mode
* `swagger movie-collection start -m`
